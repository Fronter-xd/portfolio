# Rousan Raahat Portfolio

Professional portfolio showcasing AI-powered projects with a modern cyber-dark theme.

## Projects

| Project | Description | Tech Stack |
|---------|-------------|------------|
| **Apex-Lead-Gen** | AI-powered lead generation engine | Python, PRAW, Tweepy, Llama-3 |
| **Sentinel-Private-AI** | On-premise RAG document chatbot | Next.js, LangChain, Ollama |
| **Echo-Voice-Support** | Real-time voice AI assistant | Python, Whisper, Ollama, Edge-TTS |
| **Nova-Dashboard** | Real-time analytics dashboard | Next.js, D3.js, Framer Motion |
| **Horizon-Stories** | Cinematic scroll storytelling | Next.js, GSAP, Framer Motion |
| **Stellar-3D** | 3D AI deployment globe | React Three Fiber, Zustand |

## Tech Stack

- **Next.js 14** - App Router, React Server Components
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **TypeScript** - Type-safe development

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
